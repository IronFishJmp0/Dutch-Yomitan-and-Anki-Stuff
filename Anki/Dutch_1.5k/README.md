# Dutch 1.5k
This will be a frequency based deck that covers around the 1.5k most common words in the Dutch language. Due to the fact many common words are used in many different ways, this deck will have more than 1.5k cards. A few exceptions will also be made to complete word groups (ex, 1-10 and 12 fall within the top 1.5k, but 0 and 11 don't, so they will be added as an exception.)

The goal of this deck is to have one definition of a word per card, an example sentence for the word that uses it in the way the definition suggests, and audio for the example sentence.

Audio will come from a combination of forvo audio for individual words (double checked to make sure it is the correct word), and the example sentence and audio will be found from Youglish or other sources. Where possible, attribution for these will be included in the card.

### Aggregate.py
This is a script that is used for combining frequency lists. To change what you want it to give you, you will need to edit the it, then run it without parameters.
Raw data files consumed by the script are not included in this repository.
