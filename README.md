# What is this?
This repository is a home for Dutch learning resources I have either made, converted or found. To the best of my knowledge, no AI was used in the making of this repository.

### Yomitan
These are resources for the Yomitan popup dictionary.
At time of writing, this directory just includes frequency dictionaries, but I plan to add term dictionaries down the line.

### Anki
These are resources for the Anki Spaced Repition System program.
This is where the dutch 1.5k anki deck will live while it is being created.
