This directory houses frequency dictionaries that have been converted from other formats to be useable with Yomitan.
I have attributed authorship of the dictionaries to the person or organization who initially created the frequency dictionary.

Reformat.py is a simple python program that takes in a csv or tsv file containing frequency information and outputs a frequency dictionary that can be imported into yomitan.
	For more information, run `python Reformat.py --help`

wortshcatz contains frequency dictionaries that have been converted from https://wortschatz.uni-leipzig.de/en/download/nld

OpenSubtitles contains frequency dictionaries converted from hermitdave's [FrequencyWords repository](https://github.com/hermitdave/FrequencyWords)
